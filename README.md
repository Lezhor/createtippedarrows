# Create: Tipped Arrows

[![Modrinth Downloads](https://img.shields.io/modrinth/dt/NwYNtXzG?logo=modrinth&style=flat-square&label=Modrinth%20Downloads)](https://modrinth.com/mod/createtippedarrows)
[![CurseForge Downloads](https://img.shields.io/curseforge/dt/1521802?logo=curseforge&style=flat-square&label=CurseForge%20Downloads)](https://www.curseforge.com/minecraft/mc-mods/create-tipped-arrows)

Automate Archery with **Create: Tipped Arrows**. This lightweight addon for the [Create Mod](https://www.curseforge.com/minecraft/mc-mods/create) allows you to fully automate the production, recovery, and recycling of tipped arrows using familiar mechanical processes.

---

## 📊 Project Specifications

| Feature | Details |
| :--- | :--- |
| **Minecraft Version** | 1.21.1 |
| **Loader** | NeoForge |
| **Required Dependencies** | [Create Mod](https://modrinth.com/mod/create) (0.6+) |
| **Optional Dependencies** | [JEI](https://modrinth.com/mod/jei) (Highly Recommended) |
| **Configurability** | Adjustable fluid amounts for tipping/emptying |

---

## 🛠️ Development

This project uses **NeoForge** and **Registrate** for data generation and registration.

### Building
```bash
./gradlew build
```

### Data Generation
```bash
./gradlew runData
```
